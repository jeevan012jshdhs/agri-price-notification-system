# Agricultural Product Price Notification System

A responsive web application to display and manage agricultural product prices.

## Features
- View agricultural product prices
- Admin login system
- Add product prices
- LocalStorage-based data persistence
- Responsive design
- Prices displayed in ₹

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this repository
2. Open index.html in your browser
